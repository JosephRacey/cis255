My Favorite CSS Properties

Joseph H. Racey III

Introduction The purpose of this page is to demonstrate my basic ability to use CSS to format and style a web page in the way that I like. It also serves the purpose if discussing my 5 favorite CSS porperties, demonstrating my ability to impliment media calls, showing that I can impliment animations on a page, and finally to show that I understand how Chrome devtools works.
I would have to say that the position property in my top 5 favorite CSS properties. With the position property I can position any element on a page to any location that I want. One such example would be that using the absolute position property no matter the size of the wondow my "Go to Github" form button is positioned in the very top right corner. You can also create some cool effects like a floating element that wont change its position while the user scrolls through a page with the fixed value.
My next favorite CSS property is really a combination of the color and background-color properties, I like that you can change the color of every piece of text with ease. It really helps bring attention to the text that you want people to see if it contrasts the rest of the page. I think that evey page needs some degree of customization, this is one of the main reasons that I chose these properties as one of my favorites.
The border property is another of my favorite CSS properties. I like being able to seperate unrelated sections of my page so that users get a better understanding of where one topic ends and another begins. I can also see it being used as a was to direct the users eye to the most important information on the page such as a form that needs to be completed.
I like the idea of the float property, it lets you wrap text around images and set a position of certain elements, you can use it to create columns on your page, and choose the behavior of the elements around it. Although I didn't use it here, it can be used to create multi column layouts by floating blocks to the left or right.
I don't think that any page would be complete without modifying the font in some way. I find it incredibly boring when evey page uses the same default font. It is for this reason that I have chosen the font property as one of my favorite CSS properties. I think I've seen times new roman at least a hundred times on my own papers from high school and my early English classes.
